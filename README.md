# Skaffa github-konto
För nästa del av kursen "Hello, World" behöver du skaffa ett konto här på github. Med det kan du synka med andra deltagare och hålla ditt projekt uppdaterat. 
1. Gå till [github.com/signup](https://github.com/signup)
2. Fyll i formuläret och registrera dig

Det finns flera sätt att interagera med git. Det enklaste sättet är att använda Git for Desktop.
1. Gå till [desktop.github.com/download](https://desktop.github.com/download/)
2. Ladda ned programmet och installera det.
3. Starta programmet och logga in med dina uppgfiter från tidigare

Nu när programmet är installerat ska du kopiera kursens gemensamma projekt
1. På den här sidan, klicka på den gröna knappen knappen "Code" längst upp till höger. ![image](https://github.com/user-attachments/assets/d1545bbd-a0ef-46e3-9942-0ff094b2a6ea)
2. Kopiera [länken](https://github.com/okatima/hello-world-pilot.git)
3. I *Git for Desktop* tryck på "Clone a project" och klistra in länken.
4. Som målmapp ska du spara projektet i en tom mapp någonstans på din dator där du lätt kan hitta den.

![image](https://github.com/user-attachments/assets/f477af4b-97e2-42fd-98f1-3ab8ae3d7bac)


Nu kan du i Godot importera projektet som du har kopierat. 
1. Öppna Godot och tryck på "Import" i menyn
