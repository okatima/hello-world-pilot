# Skaffa github-konto
För nästa del av kursen "Hello, World" behöver du skaffa ett konto här på github. Med det kan du synka med andra deltagare och hålla ditt projekt uppdaterat. 
1. Gå till [github.com/signup](https://github.com/signup)
2. Fyll i formuläret och registrera dig
